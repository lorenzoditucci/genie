enum gabac::ContextSelectionId
strong

Supported cabac contexts.

Enumerator
bypass 

Do not use arithmetic coding.

adaptive_coding_order_0 

Current symbol only.

adaptive_coding_order_1 

Use current + previous symbol.

adaptive_coding_order_2 

Use current + previous + before previous symbol.

enum gabac::SequenceTransformationId
strong

Supported transformations.

Enumerator
no_transform 

Do nothing.

equality_coding 

Find equal values sequentially.

match_coding 

Find larger sequence matches.

rle_coding 

Find run lengths.

lut_transform 

Remap symbols based on probability.

diff_coding 

Use differences between symbol values instead of symbols.

cabac_coding 

Entropy coding based on cabac.
