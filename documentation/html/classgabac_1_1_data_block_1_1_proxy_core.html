Detailed Description

template<typename T>
class gabac::DataBlock::ProxyCore< T >

A proxy object abstracting the get/set access using regular operators. This way it is possible to use for example range based for loops or std algorithms with iterators.

Note
Even though this is convenient you should avoid it. It's slow.
Warning
Adding or removing elements from the data block this object is referring to leads to undefined behaviour. Don't do it as long as a proxy object is existent.
